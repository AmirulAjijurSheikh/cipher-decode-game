# 🔐 Cipher Decode Game

An AI-powered browser-based puzzle game where players decode sentences by mentally unscrambling rotated and flipped letters.

## 🎮 Play Now
👉 [Play Cipher Decode](https://AmirulAjijurSheikh.github.io/cipher-decode-game/)

## 🧠 How It Works
Each letter in a sentence is randomly transformed — rotated by 45°, 90°, 180°, or flipped horizontally/vertically. The player reads the twisted letters and types the original sentence.

## ✨ Features
- 🤖 AI-generated personalized sentences using the Anthropic Claude API
- 📈 4 difficulty levels: Rookie → Code Breaker → Mind Bender → Cipher Master
- 🏆 Rank system with level-up celebrations
- 💡 Hint system and skip option
- 🎉 Confetti animations and sound effects
- 💾 Progress saved across sessions
- 📱 Mobile friendly
- 🔥 Streak tracking and accuracy stats

## 🛠️ Built With
- HTML5, CSS3, JavaScript (vanilla — no frameworks)
- Anthropic Claude API (AI sentence generation)
- Web Audio API (sound effects)
- Canvas API (confetti particles)
- Google Fonts (Space Grotesk, JetBrains Mono)
- Git & GitHub Pages
